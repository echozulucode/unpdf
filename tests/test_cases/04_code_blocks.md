# Code Blocks

## Simple Code Block

```
This is a simple code block
with multiple lines
of code.
```

## Python Code Block

```python
def hello_world():
    """Print hello world."""
    print("Hello, World!")
    
hello_world()
```

## JavaScript Code Block

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet("World"));
```
